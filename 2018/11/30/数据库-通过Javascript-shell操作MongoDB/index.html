通过Javascript shell操作MongoDB

macOs 安装MongoDB

1.更新 Homebrew的package数据库（macosx上的软件包管理工具）

$ brew update

2.安装MongoDb

$ brew install mongodb

3.启动MongoDb

$ brew services start mongodb

4.连接到mongo

$ mongo

5.关闭MongoDb

$ brew services stop mongodb
